## Note de conception — typographie française canonique (M6.1)

### Le piège : pollution du code, JSON et blocs

Une regex naïve `: ` → ` : ` (NBSP) sur le repo entier altère :

- les blocs Go : `func f() error {` ne doit pas devenir `func f()`+NBSP+`:`
- les liens Markdown : `[PRD](PRD.md)` ne doit pas voir ses `:` modifiés
- JSON : `"key": "value"` doit rester littéral, byte-identique pour préserver les golden tests (`testdata/mini-corpus.jsonl`, `testdata/golden-corpus.jsonl`, `current.json` produits par M5)
- URLs `https://…`, codes inline ` `` ` (un seul backtick)

### Stratégie retenue — passe sélective, hors blocs de code, hors liens

1. **Cibles autorisées** : `PRD.md`, `CLAUDE.md`, `PRDPlanning.md`, `README.md`, `CHANGELOG.md`, `docs/**/*.md` *(à l'exclusion de `docs/superpowers/plans/*.md` — les sous-plans antérieurs sont des artefacts immuables livrés)*.

2. **Cibles exclues explicitement** :
   - Tout fichier `.go`, `.json`, `.jsonl`, `.yml`, `.yaml`, `.toml`, `Makefile`, `LICENSE`
   - `docs/superpowers/plans/*.md` (immuables — déjà committés taggés sur un milestone)
   - Tout bloc ```` ```…``` ```` à l'intérieur d'un `.md`
   - Tout span code inline `` `…` `` à l'intérieur d'un `.md`
   - Toute URL Markdown `](http…)` ou `](path…)`
   - Toute ligne contenant un `<…>` HTML brut (tableaux Mermaid, etc.)

3. **Outil** : script Python autonome `cmd/typo-fr/main.py` *(NON committé — utilitaire dev local ; ou inline `scripts/typo-fr.py`)* à exécution unique. Logique :
   ```python
   # Pseudo-code (à raffiner en M6.1)
   for md_file in targets:
       lines = md_file.read_text().splitlines(keepends=True)
       in_code_block = False
       out = []
       for line in lines:
           if line.startswith("```"):
               in_code_block = not in_code_block
               out.append(line); continue
           if in_code_block:
               out.append(line); continue
           # Masquer spans inline code et URLs avec placeholders
           masked, restore = mask_protected(line)
           transformed = apply_typo_fr(masked)  # voir règles ci-dessous
           out.append(restore(transformed))
       md_file.write_text("".join(out))
   ```

4. **Règles de transformation** *(appliquées dans cet ordre)* :
   | # | Cherche | Remplace | Notes |
   |---|---|---|---|
   | a | `[^\s\u00A0]:(\s\|$)` | `…\u00A0:$1` | Avant `:` |
   | b | `[^\s\u00A0];(\s\|$)` | `…\u00A0;$1` | Avant `;` |
   | c | `[^\s\u00A0]\?(\s\|$)` | `…\u00A0?$1` | Avant `?` |
   | d | `[^\s\u00A0]!(\s\|$)` | `…\u00A0!$1` | Avant `!` |
   | e | `"([^"]+)"` | `« \1 »` *(avec NBSP intérieurs)* | Guillemets droits → français |
   | f | `« ` → `«\u00A0` ; ` »` → `\u00A0»` | Espaces intérieurs « … » | Idempotent |
   | g | `: //` | `://` (re-protéger URLs) | Garde-fou après règle (a) |
   | h | numéros : `XII.XII` (renvois Monographie) inchangés | — | Pas de NBSP avant `.` |

5. **Vérification** :
   - Diff `git diff --stat` après run : aucun fichier `.go|.json|.yml` modifié.
   - Re-run idempotent : second appel ne produit aucun diff.
   - Grep `[^ ]:(?:[^\s/])` *(deux-points sans NBSP devant ni `//` derrière)* sur cibles : 0 hit.
   - Relecture humaine manuelle des 5 fichiers racines (PRD/CLAUDE/PRDPlanning/README/CHANGELOG) — vérifier que les renvois `chap. III.8.X` et expressions `O(n log n)` ne sont pas cassés.

**Statut** : **Probable** que la passe automatique couvre ≥ 95 % des occurrences ; **À vérifier** au cas par cas pour les guillemets droits qui ouvrent du code inline (faux positifs possibles). Plan de repli : passe manuelle ciblée + commit suiveur de polish.

### Le piège secondaire : guillemets droits dans citations

Les chap. III.8 cités contiennent des phrases comme `"τ déplace l'instant de fixation"`. La règle (e) doit les convertir. Mais des citations Go comme `var x = "hello"` dans un bloc code *(qui n'est jamais touché)* restent intactes. Cas-limite : citations sur une **ligne sans bloc** mais dans un tableau Markdown. Vérification post-run obligatoire.
